CSV File Format

Please ensure your CSV file follows this format:

Column Name Description Required Example
name Customer's full name Yes John Smith
email Customer's email address Yes [email protected]
phone Contact phone number Yes +1-555-123-4567
status Customer status (active/inactive) No active

Sample CSV Format:

name,email,phone,status
John Smith,[email protected],+1-555-123-4567,active
Jane Doe,[email protected],+1-555-987-6543,active
Mike Johnson,[email protected],+1-555-456-7890,inactive

Important Notes:

  • The first row must contain the column headers exactly as shown above
  • All fields should be comma-separated
  • If status is not provided, it will default to "active"
  • Maximum file size: 5MB
  • Supported file format: .csv only
